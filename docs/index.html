Moderuj WhatsApp skupiny přímo z prohlížeče

Zdarma Open Source Bez API
📥 Stáhnout 📖 Dokumentace

Co umí?

🚫

Spam filtr

Detekce spamových zpráv podle klíčových slov a vzorců. Mazání po vašem potvrzení.

🤬

Filtr vulgarismů

Blokování nevhodných výrazů s předdefinovaným i vlastním seznamem slov.

⏰

Časované mazání

Označení zpráv starších než nastavený čas. Smazání po vašem potvrzení.

🤖

AI asistent

Lokální Ollama AI odpovídá na dotazy o skupině. Odpovědi zkopírujete do schránky.

🎛️

Dashboard

Přehledný ovládací panel pro správu filtrů a statistik přímo v prohlížeči.

🔒

Soukromí

Vše běží lokálně na tvém počítači. Žádná data neopouštějí tvůj prohlížeč.

Jak to funguje?

1

Stáhni a spusť server

Naklonuj repozitář a spusť lokální server jedním příkazem.

2

Přidej bookmarklet

Přetáhni bookmarklet do lišty záložek v prohlížeči.

3

Otevři WhatsApp Web

Přihlás se na web.whatsapp.com jako obvykle.

4

Nastav a moderuj

Klikni na bookmarklet, nastav filtry a potvrzuj navržené akce.

⚠️ Upozornění k WhatsApp ToS Tento nástroj využívá DOM manipulaci, kterou WhatsApp ToS výslovně nepovoluje. Žádné zprávy se neodesílají automaticky — texty se kopírují do schránky. Mazání zpráv vyžaduje vaše potvrzení kliknutím. Používání na vlastní riziko.
